# Claude Code RTL

Zero-config RTL for Claude Code. Install and it just works.

<a id="english"></a>
## English

Hebrew, Arabic, and Persian text is auto-detected per message and rendered right-to-left. English stays LTR. Code blocks, tool calls, and thinking blocks always stay LTR.

### Get Started

1. Install from the Extensions panel (search "Claude Code RTL")
2. Reload when prompted
3. Done. Start typing in Hebrew, Arabic, or Persian

### Features

- Per-message RTL detection (not the entire chat)
- Streaming support (detects RTL as Claude responds)
- Code, tools, diffs, and permissions stay LTR
- Input box auto-detects direction from first character
- Auto re-injects when Claude Code updates

### Supported IDEs

- VS Code
- Cursor
- Antigravity

### Commands

| Command | Description |
| --- | --- |
| `Claude RTL: Enable` | Enable RTL (runs automatically on install) |
| `Claude RTL: Disable` | Disable RTL and restore original files |

<a id="עברית"></a>
## עברית

תמיכה אוטומטית ב-RTL עבור Claude Code. מזהה עברית בכל הודעה ומציג אותה מימין לשמאל. אנגלית נשארת LTR. בלוקים של קוד, כלים וחשיבה תמיד נשארים LTR.

### איך מתחילים

1. התקינו מחלונית ההרחבות (חפשו "Claude Code RTL")
2. לחצו "Reload" כשתתבקשו
3. זהו. התחילו לכתוב בעברית

### יכולות

- זיהוי RTL לכל הודעה בנפרד (לא כל הצ'אט)
- תמיכה בסטרימינג (מזהה RTL תוך כדי תגובת Claude)
- קוד, כלים, דיפים והרשאות נשארים LTR
- תיבת הקלט מזהה כיוון אוטומטית לפי התו הראשון
- מתעדכן אוטומטית כש-Claude Code מתעדכן

### סביבות נתמכות

- VS Code
- Cursor
- Antigravity

### פקודות

| פקודה | תיאור |
| --- | --- |
| `Claude RTL: Enable` | הפעלת RTL (רץ אוטומטית בהתקנה) |
| `Claude RTL: Disable` | כיבוי RTL ושחזור קבצים מקוריים |

<a id="العربية"></a>
## العربية

دعم RTL تلقائي لـ Claude Code. يكتشف النص العربي في كل رسالة ويعرضه من اليمين إلى اليسار. النص الإنجليزي يبقى LTR. أكواد البرمجة والأدوات تبقى دائما LTR.

### كيف تبدأ

1. ثبت الإضافة من لوحة الإضافات (ابحث عن "Claude Code RTL")
2. اضغط "Reload" عند الطلب
3. انتهى. ابدأ الكتابة بالعربية

### المميزات

- اكتشاف RTL لكل رسالة على حدة (ليس المحادثة بأكملها)
- دعم البث المباشر (يكتشف RTL أثناء رد Claude)
- الأكواد والأدوات والأذونات تبقى LTR
- مربع الإدخال يكتشف الاتجاه تلقائيا من أول حرف
- يعيد الحقن تلقائيا عند تحديث Claude Code

### البيئات المدعومة

- VS Code
- Cursor
- Antigravity

<a id="فارسی"></a>
## فارسی

پشتیبانی خودکار RTL برای Claude Code. متن فارسی را در هر پیام شناسایی کرده و از راست به چپ نمایش می دهد. متن انگلیسی LTR باقی می ماند. بلوک های کد و ابزارها همیشه LTR هستند.

### شروع کار

1. از پنل افزونه ها نصب کنید ("Claude Code RTL" را جستجو کنید)
2. وقتی خواسته شد "Reload" را بزنید
3. تمام. شروع به نوشتن فارسی کنید

### ویژگی ها

- شناسایی RTL برای هر پیام به صورت جداگانه
- پشتیبانی از استریم (RTL را هنگام پاسخ Claude شناسایی می کند)
- کد، ابزارها و مجوزها LTR باقی می مانند
- کادر ورودی جهت را از اولین کاراکتر شناسایی می کند
- هنگام بروزرسانی Claude Code خودکار بازتزریق می شود

### محیط های پشتیبانی شده

- VS Code
- Cursor
- Antigravity
